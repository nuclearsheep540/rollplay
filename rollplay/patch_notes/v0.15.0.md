# Patch Notes v0.15.0

**Release Date:** 2025-11-18

---

## Campaign & Invite System

- Users are now invited to campaigns instead of individual game sessions
- Added campaign invite accept/decline flow
- Players auto-enrolled to all sessions when joining a campaign
- View campaigns you're participating in (not just ones you own)

## Dashboard

- Renamed "Games" tab to "Sessions"
- Reordered navigation: Campaigns, Sessions, Characters, Friends
- Default tab changed from Characters to Campaigns
- Added 5-second polling for Campaigns, Sessions, and Friends tabs
- Sessions tab is now read-only (displays active sessions only)

## Game Lobby

- DM now appears in lobby as connected user
- Initial game state sent on WebSocket connect (fixes refresh issues)
- Seats clear properly on disconnect/refresh
- DM prevented from taking player seats
- Players prevented from taking multiple seats
- Added "Back to Dashboard" button
- Session ended modal with countdown progress bar (replaces alert)
- Persistent audio across reconnects

## UI Improvements

- Delete campaign now uses modal instead of browser confirm dialog
- Campaign tiles toggle expand/collapse on click
- Landing page text simplified to "All you need is an email address"
- Username input text color fixed (was too light)
- Game status text now capitalised
- "Create Session" button uses plus icon
- Email code copy button improved

## Backend

- Fixed cascade delete for game_joined_users
- Campaign player_ids migrated from JSON to proper array column
- Added invited_player_ids field to campaigns
- Reusable InviteButton component added to shared components
