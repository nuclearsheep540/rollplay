---

## Image Display

- **NEW:** DMs can load full-screen images into the game view for visual storytelling — scene art, handouts, portraits
- **NEW:** Image selection panel with upload, campaign library browse, and global library browse
- **NEW:** "Load Image" / "Clear Image" controls in the DM drawer
- **ENHANCED:** Active display toggles between map and image — clearing one falls back to the other if available
- **ENHANCED:** Late-joiners sync to the current active image automatically

---

## SFX Soundboard

- **NEW:** 3×3 soundboard grid for instant one-shot sound effects during gameplay
- **NEW:** Each pad supports upload, campaign library pick, or global library browse
- **NEW:** Per-pad volume sliders with debounced save to server
- **ENHANCED:** SFX pads broadcast to all players in real-time via WebSocket

---

## BGM Mixer & DJ Cue System

- **NEW:** Multiple parallel BGM channels replace the single music track
- **NEW:** DJ Cue system for seamless transitions — preview (PFL), transition mode (Fade/Cut), and program (PGM) columns
- **NEW:** Per-channel effects toggles: High-Pass Filter, Low-Pass Filter, Reverb
- **NEW:** Per-channel Mute and Solo controls
- **NEW:** Adjustable fade duration (100ms–10s) with quick ±100ms buttons
- **ENHANCED:** Track config remembered when swapping assets — volume, loop state, effects, and pause position restore automatically
- **ENHANCED:** "Stop All" button kills all playback across channels instantly

---

## Asset Type Split

- **NEW:** Dedicated Music, SFX, and Image asset types replace the old unified audio type
- **NEW:** Music assets store default volume, looping, and per-track effect toggles
- **NEW:** SFX assets store default volume — always one-shot, no looping
- **ENHANCED:** Asset library filtering updated to reflect the new type categories

---

## Session Persistence

- **NEW:** Image config persisted across sessions — resume with the same scene art loaded
- **NEW:** Active display state (map vs image) saved and restored on session start
- **ENHANCED:** Audio state ETL expanded to handle multi-channel BGM and SFX pad configs

---

## Bug Fixes & Improvements

- **FIXED:** Auth token refresh now works reliably — resolved silent failures from expired access tokens
- **IMPROVED:** All dashboard hooks and mutations migrated to `authFetch` for consistent token handling
- **IMPROVED:** Backend module structure cleaned up — standardised repository and schema directory naming

---

This release introduces the Media Foundation — a unified system for images, background music, and sound effects. DMs can now set the scene with atmospheric images, layer multiple music tracks with DJ-style transitions, and trigger sound effects from a quick-fire soundboard. All media state persists across sessions and syncs to late-joiners automatically.
