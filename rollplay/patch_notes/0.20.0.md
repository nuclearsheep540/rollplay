---

## Asset Library

- **NEW:** Upload and manage maps, audio, and images from the dashboard
- **NEW:** Upload or select existing maps directly from in-game sessions
- **NEW:** Asset type filters with All, Maps, Audio, and Images views
- **NEW:** Multi-select view for bulk asset management
- **NEW:** Adjustable grid slider for asset display density
- **ENHANCED:** File type validation rejects unsupported uploads with clear feedback
- **ENHANCED:** FontAwesome icons replace emoji across asset cards and controls

---

## Session Management

- **NEW:** Spectator mode allows players to join sessions without a character
- **NEW:** Character select modal offers "Create New Character" shortcut
- **ENHANCED:** Session start shows animated loading state while preparing game resources
- **FIXED:** Character locks re-introduced to prevent duplicate selections
- **FIXED:** Character release when removing a player from a campaign

---

## Campaign Management

- **ENHANCED:** Invite modal preserves campaign state between opens
- **FIXED:** Missing user metadata when inviting via UUID input

---

## Security & Stability

- **ENHANCED:** Campaign membership validated before serving campaign assets
- **ENHANCED:** AWS S3 configuration uses explicit field definitions for clear startup errors
- **FIXED:** S3 key fallback removed — prevents silently loading broken map URLs
- **IMPROVED:** Dead code cleanup across sessions, game, and asset modules

---

This release introduces the Asset Library — a centralised place to upload, manage, and use maps, audio, and images across campaigns and live sessions. Also includes spectator mode, session UX improvements, and security hardening on asset access.
